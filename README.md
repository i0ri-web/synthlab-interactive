# 🎹 SynthLab PRO

Web Audio APIを使用してブラウザ上で動作する本格的なアナログ・シンセサイザーのシミュレーターです。
シンセサイザーの音作りの仕組みを、実際にパラメータ（ノブ）を触りながら視覚的・直感的に学ぶことができます。

🌐 **[デモをプレイする](https://i0ri-web.github.io/synthlab-interactive/)**

## ✨ 特徴 (Features)
- 🎛️ **ハードウェア風UI**: 全てのパラメータはドラッグ操作可能な「ノブ」と「縦型フェーダー」で操作します。
- 🌊 **デュアル・オシレーター**: メイン音源（Osc 1）とサブ音源（Osc 2）を重ねて分厚い音（Super Sawなど）を作れます。
- 📈 **ADSR エンベロープ**: アタック、ディケイ、サステイン、リリースで時間経過による音量・音色変化を作れます。
- 🎚️ **LFO & Filter**: 音やフィルターに自動的な揺らぎ（ビブラートやワウワウ）を与えたり、特定帯域をカットして音色を整えます。Env Depthによりフィルターも時間変化させられます。
- 🪞 **Delay FX**: 「やまびこ効果」で音に空間的な反響をもたせます。
- 📺 **Waveform Visualizer**: オシロスコープ機能を用いて出力される波形をリアルタイムに視覚化します。

## 🚀 使い方 (How to Use)

### 1. 音を鳴らす
- 画面右下の **「HOLD TO PLAY」** ボタンをマウスでクリック（長押し）するか、スマートフォンなどでタップし続けると音が鳴ります。
- **PCキーボード演奏**: ご利用のPCのキーボード `A, W, S, E, D, F, T, G, Y, H, U, J, K` （ピアノの白鍵・黒鍵に相当）を押すことで、和音を含む実際の演奏が可能です。

### 2. パラメータの操作（ノブの使い方）
- このアプリは、本物のアナログシンセサイザーに近い操作感を再現しています。
- 丸い **「ノブ」** の上にマウスカーソル（または指）を合わせ、**クリックしたまま「上・下」にドラッグ** すると、数値をなめらかに上げ下げすることができます。
- 波形（Waveform）や揺らす対象（Target）などの名称部分は、クリックすることでドロップダウンリストから選択できます。
- 画面右下のマスターボリュームは **「縦型フェーダー」** になっています。スライダーのつまみ部分を上下にドラッグして全体の音量を調整してください。

## 🛠️ 主なモジュールの解説・音作りのヒント
* **Oscillator 1 & 2**: 音の元となる波形（サイン波、矩形波、ノコギリ波、三角波）を選びます。Osc 2の「Octave」を `-1` にし、「Detune」を `10 ~ 15` 程度ズラすと、太くて広がりのあるリード・ベースサウンドになります。
* **Envelope ADSR**: 鍵盤を押した瞬間から、離した後の音の減衰の仕方を決めます。A（Attack）を長くするとバイオリンのようにフワッと鳴り、R（Release）を長くするとピアノのペダルのような余韻が残ります。
* **Filter**: 指定した周波数（Cutoff）より上（Lowpass）、または下（Highpass）の音を削り取ります。「Reso (Resonance)」を上げると削り目にクセがつき、「ミョオン」と鳴くような派手なシンセ音になります。
* **LFO**: 自動で揺れる波を作り、ピッチ（音程）にかけてビブラートを作ったり、フィルターにかけてワウワウ効果を作れます。Rateが揺れの速さ、Depthが深さです。
* **Delay**: 音に「やまびこ」のような反響音を足してリッチなサウンドにします。
